\section{实验目的}
\begin{enumerate}[label=\arabic*. ]
  \item 了解热传导现象、热传导定律和导热系数的物理意义；
  \item 学习用稳态法测量非良导体导热系数的基本思路，体会参量转换法在实验设计中的作用；
  \item 掌握利用散热盘冷却速率间接求取样品导热系数的方法。
\end{enumerate}

\section{实验原理}

\subsection{热传导定律}
当物体内部各处温度不均匀时，热量会从高温区域向低温区域传递，这种现象称为热传导。若热量沿样品厚度方向 $z$ 传输，在垂直于传热方向的微元截面 $\mathrm{d}S$ 上，傅里叶热传导定律可写为
\[
  \frac{\mathrm{d}Q}{\mathrm{d}t}
  =
	-\lambda \mathrm{d}S \frac{\mathrm{d}T}{\mathrm{d}z},
\]
其中 $\lambda$ 为材料的导热系数，$\mathrm{d}S$ 为垂直于传热方向的截面积，负号表示热流方向与温度升高方向相反。由此可见，导热系数的物理意义是：当温度梯度为单位值时，单位时间内垂直通过单位截面积的热量。

利用上式测量材料导热系数时，关键在于解决两个问题：一是如何在样品内建立并确定温度梯度，二是如何测得通过样品的传热速率。本实验采用稳态法与参量转换法来分别解决这两个问题。

\subsection{稳态法测量导热系数}
将待测硅橡胶样品 $B$ 夹在加热盘 $C$ 与散热盘 $A$ 之间。稳定加热后，上下两盘分别维持近似恒定温度 $T_1$ 与 $T_2$，样品内部便建立起沿厚度方向的稳定温度梯度。若样品厚度为 $h_B$、半径为 $R_B$，且满足 $h_B \ll 2R_B$，则侧面散热相对很小，可近似认为热量仅沿垂直于样品表面的方向传递；又由于铜盘导热性能远好于样品，在稳态下可近似认为同一铜盘各处温度相同，于是样品内部的温度梯度可写为
\[
  \frac{\mathrm{d}T}{\mathrm{d}z}
  \approx
  \frac{T_1 - T_2}{h_B}.
\]

稳态时，加热盘供给的热量、样品内部的传热量与散热盘向环境散失的热量三者相等，因此样品中的传热速率就等于散热盘 $A$ 在稳态温度 $T_2$ 下的散热速率。这样，原本难以直接测量的样品传热速率，便转化为测量散热盘的散热过程。

\subsection{散热速率的参量转换}
设散热盘 $P$ 的质量为 $m$，比热容为 $c$，其温度在自然冷却过程中随时间变化为 $T(t)$。根据热容量关系，散热盘在温度 $T_2$ 附近的散热速率可表示为
\[
  \left.\frac{\mathrm{d}Q}{\mathrm{d}t}\right|_{T=T_2}
  =
  -mc
  \left.\frac{\mathrm{d}T}{\mathrm{d}t}\right|_{T=T_2}.
\]
式中负号表示冷却过程中散热盘温度随时间下降。实验上先使散热盘温度略高于稳态温度 $T_2$，再记录其自然冷却曲线，在 $T=T_2$ 附近取切线斜率，即可求得该温度下的冷却速率。

但应注意，上式对应的是散热盘全部表面均暴露于空气中时的冷却过程；而在稳态测量中，散热盘上表面与样品接触，被覆盖部分不再直接向空气散热，因此必须做面积修正。若散热盘 $P$ 的半径和厚度分别为 $R_P$ 与 $h_P$，则其自然冷却时的总散热面积为
\[
  2\pi R_P^2 + 2\pi R_P h_P,
\]
而稳态测量时的有效散热面积为
\[
  \pi R_P^2 + 2\pi R_P h_P.
\]
稳态时散热盘的实际散热速率应修正为
\[
  \frac{\mathrm{d}Q}{\mathrm{d}t}
  =
  -mc
  \frac{R_P + 2h_P}{2R_P + 2h_P}
  \left.\frac{\mathrm{d}T}{\mathrm{d}t}\right|_{T=T_2}.
\]

根据前面的分析，修正后的散热速率就是样品在稳态时的传热速率。再结合傅里叶热传导定律，并取传热截面积
\[
  S=\pi R_B^2,
\]
可得待测样品导热系数的计算式
\[
  \lambda
  =
  -mc
  \frac{R_P + 2h_P}{2R_P + 2h_P}
  \frac{1}{\pi R_B^2}
  \frac{h_B}{T_1 - T_2}
  \left.\frac{\mathrm{d}T}{\mathrm{d}t}\right|_{T=T_2}.
\]
本实验中铜盘比热容取
\[
  c = 0.385 \times 10^3\ \mathrm{J\,kg^{-1}\,^{\circ}C^{-1}}.
\]

\section{实验内容和步骤}

\noindent\textbf{1.} 按照实验接线图连接装置，打开热学综合实验平台电源，在实验目录中选择“非良导体导热系数实验”，并拖出温度计 I、温度计 II 和计时器模块。

\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.86\linewidth]{assets/image9.png}
\end{figure}

\noindent\textbf{2.} 记录样品和散热盘的几何参数，包括硅橡胶样品厚度 $h_B$、半径 $R_B$，散热盘厚度 $h_P$（仪器软件中分别记为 $R_A$ 和 $h_A$）、半径 $R_P$ 以及散热盘质量 $m$。

\noindent\textbf{3.} 将硅橡胶样品夹在加热盘 A 与散热盘 P 之间，调节支架上的微调螺丝，使样品 B 与上下铜盘 A、P 接触良好；将两支温度探头分别插入对应铜盘侧孔并与铜盘充分接触。A 为温度通道 I，B 为温度通道 II。

\noindent\textbf{4.} 设置控温器温度为 $80\,^{\circ}\mathrm{C}$，启动温度记录功能，系统每隔 $60\,\mathrm{s}$ 自动记录一次加热盘温度 $T_1$ 与散热盘温度 $T_2$。

\noindent\textbf{5.} 待加热一段时间后（约40分钟），当 $T_1$ 读数基本稳定（波动小于0.2℃），再每隔 $2\,\mathrm{min}$ 记录一次 $T_1$ 和 $T_2$ 于表 \ref{table:steady}，直到 $T_2$ 也基本稳定（10分钟内波动小于0.2）。

\noindent\textbf{6.} 测量散热盘在稳态温度 $T_2$ 附近的散热速率：移开A，取下B，再使P与A直接接触，当P的温度上升到高于稳态 $T_2$ 的值约10℃左右时，再将A移开，让P的所有表面均暴露于空气中。使P自然冷却，并开始记录散热温度与时间入表 \ref{table:cooling} 中，直到温度下降到$T_2$以下一定值，停止记录。

\noindent\textbf{7.} 根据冷却曲线在 $T=T_2$ 附近的切线斜率求出冷却速率，再代入导热系数公式，计算样品的导热系数。

\section{数据处理}
\subsection{稳态温度}
表 \ref{table:steady} 中记录值变化平缓，未见明显离群点，因此直接取平均值作为稳态温度。由
\[
  \overline{T_1}
  =
  \frac{1}{10}\sum_{i=1}^{10} T_{1i}
  =
  \frac{79.6\times 4 + 79.5\times 6}{10}
  =
  79.54\,^{\circ}\mathrm{C},
\]
\[
  \overline{T_2}
  =
  \frac{1}{10}\sum_{i=1}^{10} T_{2i}
  =
  \frac{588.0}{10}
  =
  58.80\,^{\circ}\mathrm{C}
\]
可得稳态温差
\[
  \Delta T = \overline{T_1} - \overline{T_2} = 20.74\,^{\circ}\mathrm{C}.
\]

\subsection{冷却曲线与冷却速率}
根据表 \ref{table:cooling} 中的自然冷却数据作散热盘冷却曲线，如图 \ref{fig:cooling-fit} 所示。对全部冷却数据采用经验函数
\[
  T = \frac{a}{t+b}+c
\]
进行曲线拟合，得到
\[
  T
  =
  \frac{8.779\times 10^4}{t+1331.56} + 3.789,
  \qquad
  R^2 = 0.9976.
\]
其导函数为
\[
  \frac{\mathrm{d}T}{\mathrm{d}t}
  =
  -\frac{8.779\times 10^4}{(t+1331.56)^2}.
\]
当 $T=\overline{T_2}=58.80\,^{\circ}\mathrm{C}$ 时，由上式可解得
\[
  t = \frac{8.779\times 10^4}{58.80-3.789} - 1331.56 = 264.24\,\mathrm{s}.
\]
将该时刻代入导函数，得到稳态温度附近的冷却速率
\[
  \left.\frac{\mathrm{d}T}{\mathrm{d}t}\right|_{T=\overline{T_2}}
  =
  -0.03447\,^{\circ}\mathrm{C}\,\mathrm{s}^{-1}.
\]

\subsection{导热系数计算}
将
\[
  m = 0.823\,\mathrm{kg},\quad
  h_P = 0.007\,\mathrm{m},\quad
  h_B = 0.005\,\mathrm{m},\quad
  R_P = R_B = 0.065\,\mathrm{m},
\]
\[
  c = 385\,\mathrm{J\,kg^{-1}\,^{\circ}C^{-1}},\quad
  \Delta T = 20.74\,^{\circ}\mathrm{C},\quad
  \left.\frac{\mathrm{d}T}{\mathrm{d}t}\right|_{T=\overline{T_2}}
  =
  -0.03447\,^{\circ}\mathrm{C}\,\mathrm{s}^{-1}
\]
代入导热系数公式，得
\[
  \lambda
  =
  -0.823 \times 385
  \times \frac{0.065 + 2\times 0.007}{2\times 0.065 + 2\times 0.007}
  \times \frac{1}{\pi \times (0.065)^2}
  \times \frac{0.005}{20.74}
  \times (-0.03447),
\]
\[
  \lambda = 0.109\,\mathrm{W\,m^{-1}\,K^{-1}}.
\]
该结果处于非良导体常见的 $10^{-1}\,\mathrm{W\,m^{-1}\,K^{-1}}$ 量级，说明待测硅橡胶样品导热性能较弱，具有较好的隔热特征。

\section{实验的分析和总结}
\begin{enumerate}[label=\arabic*. ]
  \item 由实测数据计算得样品导热系数
  \[
    \lambda = 0.109\,\mathrm{W\,m^{-1}\,K^{-1}},
  \]
  表明该样品属于导热性能较差的非良导体，热量通过其厚度方向传递较慢，作为隔热材料是合理的。
  \item 本实验的关键设计是先用稳态法在样品两侧建立稳定温差，通过测得 $T_1$、$T_2$ 和样品几何尺寸确定温度梯度；再利用参量转换法，把难以直接测量的传热速率转化为散热盘在同一温度附近的冷却速率，从而将导热系数测量化为温度和时间的测量。
  \item 实验误差主要来自以下几方面：一是稳态测温阶段中 $T_2$ 仍有缓慢上升，说明系统虽已接近稳态但未达到理想静止状态，这会使最终结果偏小；二是样品与铜盘之间若存在接触不紧密或接触热阻，会增大测得的温差，从而低估导热系数；三是冷却曲线虽然采用了全数据拟合，但拟合模型的选取、温度读数分辨率以及环境对流条件变化仍会影响在 $\overline{T_2}$ 处求得的导数，从而带来一定系统误差。
\end{enumerate}
